All the information linked to a particular user can be viewed in User profile details page. You can click any user profile from User listing page to land in the user details page.

User profile details has 4 sections

1. User information
2. User Properties
3. Notifications Sent
4. Preferences

# User information

In the User details page, you will see the User information section on the left hand side as shown below. This has 2 sections

1. General - contains Name and Timezone of the user.
2. Channels Configured - All the channel information available for that particular user.

You can edit these property values by clicking on the 'Edit' icon.
![](https://files.readme.io/aeed0a5-User_Info.png)

# User Property

1. In the User details page, you will see the User property section on the left hand side as shown below. All the custom properties of the user will be available here.
   ![](https://files.readme.io/00e987d-User_properties.png)

2. You can add any new property by searching for it
   ![](https://files.readme.io/46409a7-user_prop_wishlist.png)

3. Click 'Add new property'. You will see the list of datatypes.
   ![](https://files.readme.io/6dbb572-choose_datatype.png)

4. Choose appropriate data type and enter value for the new property and save it.

You can also edit any property value manually by clicking the 'Edit' icon.

# Notifications sent

All the notifications sent to the user can be seen under the 'Notification sent' tab.
![](https://files.readme.io/cabe381-Notif_Sent.png)

You can filter using User status, Channels or Date Range. You will be able to see the notifications sent data for the last 7 days only. Clicking on any notification sent will show you the detailed sent logs popup.

# Preferences

1. All the sections and subscription topics linked to the user will be shown here.
   ![](https://files.readme.io/a150dd9-Preferences.png)

2. You will see the section, subscription topic, channel's default status and the preference of the user.
3. You can filter using Channel status and user preference.
