# How to add a new user property from user listing page

1.  To add a new global user property, click on the Properties list CTA at the top right corner of User Profile page. You will see the properties list as shown
    ![](https://files.readme.io/dd59f5c-Properties_list.png)

2.  Click 'ADD NEW'. You will see the datatype of the property you want to add. You can select one of the datatypes

        1. Date
        2. String
        3. Boolean
        4. List
        5. Number

    ![](https://files.readme.io/6b9ab6d-add_new_prop.png)

3.  For example, if you want to add a list property, then Click List. It will open a popup as shown
    ![](https://files.readme.io/dc26529-add_new_list_prop.png)

4.  Enter a unique Property name and click 'Create Property'.
5.  Now a new user property with the datatype List is added to all the users.
6.  If you want to see the new property as part of the User Profile table, then enable the toggle in 'See in Table' column.

# How to add a new user property from user profile page

1. Go to any user profile to which you want to add a user property and value from User listing page.
2. To add a new global user property, say, 'wishlist', search for it as shown under User Properties as shown
   ![](https://files.readme.io/b5f931a-user_prop_wishlist.png)

3. Click 'Add new property "Wishlist".
4. Choose the datatype of the user property.
   ![](https://files.readme.io/05b6d69-choose_datatype.png)

5. For example, if the data type you chose is 'List', then add the values as shown.
   ![](https://files.readme.io/3c37cf3-add_items.png)

6. Click 'Create Property' to add this property to all user profiles.
