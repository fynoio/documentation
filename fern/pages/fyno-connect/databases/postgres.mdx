To connect Fyno Connect with your Postgres database:

<Steps>
  <Step>Navigate to **Databases** and click on Postgres.</Step>
  <Step>In the pop-up, enter a name for the new connection and paste the MongoDB connection string or URL.</Step>
  <Step>Click on "Add Database" to test the connection and successfully save it. After the database is added successfully, you will see it listed on the same page under **Connected DBs**.</Step>
  <Step>You can now create cohorts from the data in your database. See [Create Cohorts.](./create-cohort)</Step>
</Steps>

<Tip title="Make sure your Postgres connection string is on point!">The standard connection string format looks like: `postgresql://[user\[:password\]@][netloc][:port][/dbname][?param1=value1&...]`</Tip>
