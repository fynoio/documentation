Fyno offers both Free and Paid plans for our users.

In Fyno's Free plan, you will have restrictions on how many notifications you can send per day. Also, few advanced features like Security, URL Tracking etc are not available in the Free Plan.

| S.No | Feature               | Limit                   |
| :--- | :-------------------- | :---------------------- |
| 1    | Notifications         | 500 per day             |
| 2    | Templates             | 10                      |
| 3    | Notification Events   | 10                      |
| 4    | API Keys              | 2                       |
| 5    | Single-channel Route  | 1 per channel           |
| 6    | Omni-channel Route    | 1                       |
| 7    | Team                  | Maximum 5 (incl. Owner) |
| 8    | Workspaces            | 1                       |
| 9    | Components            | 2                       |
| 10   | Workflow              | 1                       |
| 11   | Suppression List      | 10K destinations        |
| 12   | User Profiles         | 1K                      |
| 13   | User Properties       | 5 Custom properties     |
| 14   | AllowList URL         | 2                       |
| 15   | Logs storage          | 3 days                  |
| 16   | Workflow Logs storage | 1 day                   |

## Features which are not available in Free plan

1. Data Masking
2. Data Storage
3. Approval Requests
4. Fyno Shorty
5. Push Token Cleanup
6. Report Extraction
7. Conversation Logs
8. Anomaly Detection
9. Workflow Failure Alerts
10. Do Not Disturb Configuration
11. Marketing capping frequency
