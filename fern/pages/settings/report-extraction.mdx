Report extraction enables you to download Sent Logs for both Live and Test environments. Reports can be extracted for yesterday, the last 7 days, the last 28 days or a custom date range (within the last 6 months).

## Steps to extract report

1. Click on Settings Icon on the Lower Left Navigation Pane
   ![](https://files.readme.io/0863c2ef093996a4ed45d4eae5f5d71a72e8da3a5cd0761b08bfff08ecc2e4f5-image.png)

2. Navigate to Workspace Settings -> Add On's
   ![](https://files.readme.io/60da9ea-Report_Extraction.png)

3. Click 'Request Report' within the Report Extraction card.
4. Choose the version and date range , then click on 'Confirm.'

**Version**

-   Test
-   Live

**Date Range **

-   Yesterday
-   Last 7 days
-   Last 28 days
-   Custom (you can select any 30 days within the last 6 months)

4. You will get an InApp notification with the message "Your report request xxxxxx is processing!" as shown below.

5. Upon the completion of your report, an InApp notification will be sent, along with an email containing a 'Download Report' call-to-action..

![](https://files.readme.io/d14216f-InApp_Downloade.png)

6. Upon clicking 'Download Report,' a popup will appear, as shown below.

![](https://files.readme.io/b11095b-InApp_DownloadLink.png)

7. Upon clicking the download link, the report will start downloading.
