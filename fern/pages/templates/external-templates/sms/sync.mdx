## How to sync SMS templates to Fyno

> ðŸ“˜ Currently, Fyno supports creating and submitting SMS templates for DLT approval only for Vodafone Idea.

To sync SMS templates to Fyno, click sync icon next to 'Create+' button. You will see the below popup

![](/images/7e1a02b-SMS_Template_sync.png)

1. Select 'SMS' under channel
2. Select DLT provider name from where you want to sync the SMS templates.
3. Upload CSV file which is downloaded from DLT platform. Make sure you upload the file as it is, without making nay changes to the file.
4. Click 'Sync'.
5. All the SMS templates present in the CSV file will be synced to your Fyno workspace now.
6. You can see all the Synced templates in External Templates page under SMS tab.
