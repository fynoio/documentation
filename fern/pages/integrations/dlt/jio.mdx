Before we get started, you will need to have a Jio DLT account already set up.

1. Navigate to Integrations -> DLT.
2. Click on the "Jio" button.
3. In the below pop-up, fill in:
   ![](/images/dlt_jio_integration.png)

    1. **Custom name**: Provide a name that would help you identify the configured account in Fyno's portal.
    2. **DLT Provider**: 'Jio' is pre-filled.
    3. **PE Id (Private Entity Id)**: Go to your profile and get the PE id as shown below.
       ![](/images/dlt_jio_config.png)
