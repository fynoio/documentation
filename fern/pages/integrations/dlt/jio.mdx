Before we get started, you will need to have a Jio DLT account already set up.

<Steps>
### Find the Provider
Navigate to Integrations -> DLT. Click on the "Jio" button.

### Configure the Integration

In the below pop-up, fill in:
![](/images/dlt_jio_integration.png)

- **Custom name**: Provide a name that would help you identify the configured account in Fyno's portal.
- **DLT Provider**: 'Jio' is pre-filled.
- **PE Id (Private Entity Id)**: Go to your profile and get the PE id as shown below.
  ![](/images/dlt_jio_config.png)

### Complete the Integration

Click on "Add Account" once done, and you are all set!

</Steps>
