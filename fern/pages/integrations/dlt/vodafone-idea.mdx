Before we get started, you will need to have a Vodafone Idea DLT account already set up.

1.  Navigate to Integrations -> DLT.
2.  Click on the "Vodafone Idea" button.
3.  In the below pop-up, fill in:
    ![](/images/dlt_vodafone_integration.png)

        1. **Custom name**: Provide a name that would help you identify the configured account in Fyno's portal.
        2. **DLT Provider**: 'Vodafone Idea' is pre-filled.
        3. **PE id**: Username or Private Entity Id (PE Id) to login into Vodafone Idea DLT platform. You can get it from Registration APIs -> API Credentials -> User name (PE Id)
        ![](/images/dlt_vodofone_config.png)

        4. **DLT Headers**: DLT Headers must be 6 characters length.
        5. **Username**: Username is similar to Private Entity Id (PE Id).
        6. **API Password**: Go to Registration APIs -> API Credentials -> Enable API. You will be redirected to payment page, where you need to pay 10K (per year) for using this API. Once its done, you will see API Password which you need to configure in the Integration.
