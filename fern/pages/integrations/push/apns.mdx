# Prerequisites for Integration with Fyno

Before proceeding, please make sure you have the following:

- A Mac computer with the latest version of Xcode installed
- An Apple Developer account (Individual or Organisation)
- A registered iOS device for testing

### Setting up an Apple Developer Account

<Steps>
  <Step>Go to the Apple Developer website.</Step>
  <Step>Click "Account" in the top right corner.</Step>
  <Step>Sign in with your Apple ID, or create a new one if necessary.</Step>
  <Step>Enrol in the Apple Developer Program by following the on-screen instructions. Note that there is an annual membership fee.</Step>
  <Step>Once enrolled, sign in to your Apple Developer account.</Step>
</Steps>

### Generating a Private Key, Team ID, and Key ID

<Steps>
  <Step>Sign in to your Apple Developer account.</Step>
  <Step>In the left-hand menu, select "Certificates, Identifiers & Profiles."</Step>
  <Step>Under "Keys," click the "+" button to create a new key.</Step>
  <Step>Enter a descriptive name for the key, enable "Apple Push Notifications service (APNs)," and click "Continue."</Step>
  <Step>Review the details, click "Register," and then click "Download" to save the private key (.p8 file) to your computer. Important: This is the only time you can download this key, so store it in a safe place.</Step>
  <Step>Note your Team ID and Key ID from the respective columns in the "Keys" section.</Step>
</Steps>

### Configuring Application with Fyno

<Steps>
  <Step>Sign up for a Fyno account or sign in if you already have one.</Step>
  <Step>Navigate to the Integration page and under Push select "APNs".</Step>
  <Step>Give a custom integration name of your choice.</Step>
  <Step>Selection between Production/Development Environment as per your current application state.</Step>
  <Step>Copy/paste the downloaded private key (.p8 file) from the previous section.</Step>
  <Step>Enter your Team ID and Key ID in their respective fields.</Step>
  <Step>Enter your APP Bundle ID which you can find in xcode. (Open your project with Xcode, select the top project item in the project navigator at the left. Then select TARGETS -> General. Bundle Identifier is found under Identity.)</Step>
  <Step>Click "Save" to complete the configuration.</Step>
</Steps>

### Adding Fyno SDK to Your iOS App

Follow the official FYNO documentation to add the Fyno SDK to your iOS app and
configure it with your Fyno App ID.
