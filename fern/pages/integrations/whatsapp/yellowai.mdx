Before we get started, you will need an active WhatsApp number as well as templates pre-configured in your Yellow.ai account.

<Steps>
### Find the Provider
Navigate to the Integrations page. Under "SMS" section, click on the "Yellow.ai" button.

### Configure the Integration

In the pop-up, fill in:

-   **Custom name**: Provide a name to help you identify the configured account in Fyno's portal.
-   **Bot Id**: To obtain the bot ID, you can click on the web link from any page of the bot after creating it and copy the 14-digit alphanumeric value (e.g., x16763649\*\*). The bot ID is different in each environment.
    ![](/images/whatsapp_yellow_botid.png)
-   **API Key**: You can access API Key from your Yellow.ai account. Go to _Bot Settings -> API Keys_ . If there are no API keys, you can create one by clicking 'Generate API key'
    ![](/images/whatsapp_yellow_apikey.png)

-   **Sender**: Choose the WhatsApp business number from which you want to send the campaign

### Complete the Integration

Click on "Add Account" once done, and you are all set!

</Steps>

## Message Delivery Status

Yellow.ai, by default, provides the status of the WhatsApp messages without any additional setup needed.

Once the above integration is complete, the status of the notification, which will be shared by Yellow.ai will be displayed in the logs, when you click on it, under the **[Delivery](./log-details-delivery)** tab.
