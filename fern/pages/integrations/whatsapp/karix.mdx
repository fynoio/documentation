Before we get started, you will need an active WhatsApp number as well as templates pre-configured in your Karix account.

1. Navigate to Integrations.
2. Under "WhatsApp" section, click on the "Karix" button.
3. In the pop-up, fill in:
    1. **Custom name**: Provide a name to help you identify the configured account in Fyno's portal.
    2. **API/Account Key**: Go to [Power APIs](https://karix.solutions/lounge/sign_in.php) page. Click on "Create keys" button. Enter the "key name" and "description" in the create keys modal and save. Copy the API key generated by clicking on the "eye" button and add it here.
    3. **From WhatsApp Number**: Enter the registered WhatsApp number that you would like to use to initiate conversations.  
       Login to [Karix managebot ](https://managebot.karix.solutions/bot_builder_gui/#/login)account and follow the below steps:
        1. Go to "WhatsApp Campaign" -> "Profiles" page on the left navigation menu
        2. If you have already added a WhatsApp number, you'll see Mobile number and Sender Name on the right side of the page or
        3. You can add a number by clicking on "Add WhatsApp Number" button
4. Click on "Test Integration" once done. If the account is successfully connected, you will see a success message.

## Message Delivery Status

One of the platform advantage of using Fyno as a central communication system is that it shows notification analytics for all channels in your Fyno account together.

Login to [Karix managebot](https://managebot.karix.solutions/bot_builder_gui/#/login) account and follow the below steps to setup webhook URL:

1. Go to "Whatsapp Campaign" -> "Webhook_Config" page on the left navigation menu
2. Under "Delivery rule" tab, click 'Create New'.
    1. Add Callback URL given in the Karix Integration popup in the URL field.
3. Header: Key - "Content-Type", Value - "application/json"
4. Select all required delivery events for tracking
