Before we get started, you will need an active WhatsApp number as well as templates pre-configured in your Karix account.

<Steps>
### Find the Provider
Navigate to the Integrations page. Under "WhatsApp" section, click on the "Karix" button.

### Configure the Integration

In the pop-up, fill in:

-   **Custom name**: Provide a name to help you identify the configured account in Fyno's portal.
-   **API/Account Key**: Go to [Power APIs](https://karix.solutions/lounge/sign_in.php) page. Click on "Create keys" button. Enter the "key name" and "description" in the create keys modal and save. Copy the API key generated by clicking on the "eye" button and add it here.
-   **From WhatsApp Number**: Enter the registered WhatsApp number that you would like to use to initiate conversations.  
     Login to [Karix managebot](https://managebot.karix.solutions/bot_builder_gui/#/login) account and follow the below steps:
    <Steps>
        <Step>
            Go to "WhatsApp Campaign" -> "Profiles" page on the left navigation
            menu
        </Step>
        <Step>
            If you have already added a WhatsApp number, you'll see Mobile
            number and Sender Name on the right side of the page or
        </Step>
        <Step>
            You can add a number by clicking on "Add WhatsApp Number" button
        </Step>
    </Steps>

### Complete the Integration

Click on "Add Account" once done, and you are all set!

</Steps>

## Message Delivery Status

One of the platform advantage of using Fyno as a central communication system is that it shows notification analytics for all channels in your Fyno account together.

Login to [Karix managebot](https://managebot.karix.solutions/bot_builder_gui/#/login) account and follow the below steps to setup webhook URL:

<Steps>
    <Step>
        Go to "Whatsapp Campaign" -> "Webhook_Config" page on the left
        navigation menu
    </Step>
    <Step>Under "Delivery rule" tab, click 'Create New'. Add Callback URL given in the Karix Integration popup in the URL field.</Step>

    <Step>Header: Key - "Content-Type", Value - "application/json"</Step>
    <Step>Select all required delivery events for tracking
        ![](/images/whatsapp_karix_callback.png)</Step>

</Steps>
